# Документация
## Используемые библиотеки, модули, фреймворки
- os
Модуль os предоставляет множество функций для работы с операционной системой, причём их поведение, как правило, 
не зависит от ОС, поэтому программы остаются переносимыми. В частности в проекте используется os.path - модуль, 
реализующий некоторые полезные функции на работы с путями.

- flask
Это небольшой и легкий веб-фреймворк, предлагающий полезные инструменты и функции для облегчения процесса создания 
веб-приложений. Он обеспечивает гибкость и позволяет создать веб-приложение быстро, используя только один файл Python. 
Это расширяемая система, которая не обязывает использовать конкретную структуру директорий и не требует сложного 
шаблонного кода перед началом использования.

- requests
Библиотека requests является стандартным инструментом для составления HTTP-запросов в Python. Простой и аккуратный API 
значительно облегчает трудоемкий процесс создания запросов.

- keras
Keras — это высокоуровневый API, который не заостряет внимание на вычислениях на бэкенде. Содержит методы для 
конвертации изображений в массивы NumPy. Используется для дополнения данных в реальном времени.
